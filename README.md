# Project Salt Box — FOIA Archive

This repository holds the Freedom of Information Act (FOIA) requests filed by **Project Salt Box** and the records we get back from federal, state, and local agencies. Our reporting focuses on ICE detention expansion, government contracting, and immigration enforcement infrastructure, and the documents here are the raw material behind that work.

We keep these files public so that other reporters, researchers, and members of the public can read the same records we do, check our work, and build on it.

## What's in here

- **Requests** — copies of the FOIA requests we've sent, so you can see exactly what we asked for and when.
- **Responses** — the documents agencies released to us, along with response letters, fee notices, denials, and appeals.
- **Logs** — a running index of requests, the agency each one went to, the date filed, and its current status.

## How it's organized

Documents are grouped by agency, then by individual request, so everything tied to one request lives in one place.

## How to use it

You can:

- **Browse** the folders directly on GitHub to read documents in your web browser.
- **Download** a single file by opening it and clicking the download button, or grab the whole archive at once with the green **Code** button at the top of the page.
- **Search** the repository using GitHub's search bar to find a keyword, agency name, or contract number across all the files.
- **Cite** a document by linking to its page here. Every file has a permanent URL.

If you're a journalist or researcher and want to reference these records, please feel free. A link back to Project Salt Box is appreciated but not required.

## A note on the documents

These are government records released to us, and we share them as we received them. Where a release contained personal information about private individuals that the agency failed to redact, we remove it before posting. If you spot something that should have been redacted, please let us know (see below).
